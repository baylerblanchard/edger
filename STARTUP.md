# Startup Instructions

## Prerequisites
- Node.js (Frontend)
- Ruby (Backend)
- PostgreSQL (Database)

## Commands

### 1. Backend (Rails)
Run this in the `backend` directory. We use port 3001 to avoid conflicts with the frontend.
```bash
cd backend
bin/rails s -p 3001
```

### 2. Frontend (Next.js)
Run this in the root directory.
```bash
npm run dev
```

### Access
- Application: http://localhost:3000
- API: http://localhost:3001
